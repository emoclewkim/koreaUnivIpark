# koreaUnivIpark
## 고려대학교 세종캠퍼스 피트니스센터 이용 APP

기존 피트니스 센터 이용시 문제점
- 출입시 수기로 명부작성으로 인한 개인정보노출 위험과 불편함
- 코로나19로 인한 센터 이용인원 제한에 따른 원격으로 실시간인원파악 필요
